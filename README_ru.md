# Date & Time UI

Минималистичный UI эксперимент, представляющий дату и время в виде объекта в стиле кода. Исследует чистую обработку JavaScript Date и типографику без фреймворков.

## 🎨 О Проекте

Этот экспериментальный UI концепт отображает текущую дату и время в коде-подобном формате, фокусируясь на чистом управлении состоянием, потоке рендеринга и типографической эстетике.

## ✨ Возможности

- **Часы в Реальном Времени** — Живые обновления даты и времени
- **Отображение в Стиле Кода** — Моноширинное, объектно-подобное представление
- **Vanilla JavaScript** — Без фреймворков и библиотек
- **Минималистичный Дизайн** — UI с фокусом на типографику
- **Vite Build** — Быстрая разработка и оптимизированные сборки

## 🛠️ Технологии

- **Vanilla JavaScript** — Чистый JS, без фреймворков
- **HTML5** — Семантическая разметка
- **CSS3** — Типографическая стилизация
- **Vite** — Инструмент сборки и dev-сервер

## 🚀 Начало Работы

```bash
npm install
npm run dev
```

Откройте `http://localhost:5173`
