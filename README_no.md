# Date & Time UI

Et minimalt UI-eksperiment som presenterer dato og tid som et kode-stil objekt. Utforsker ren JavaScript Date-håndtering og typografi-drevet design uten rammeverk.

## 🎨 Om

Dette eksperimentelle UI-konseptet viser gjeldende dato og tid i et kode-lignende format, med fokus på ren tilstandshåndtering, render-flyt og typografi-drevet estetikk.

## ✨ Funksjoner

- **Sanntidsklokke** — Live dato- og tidsoppdateringer
- **Kode-stil Display** — Monospace, objekt-lignende presentasjon
- **Vanilla JavaScript** — Ingen rammeverk eller biblioteker
- **Minimalt Design** — Typografi-fokusert UI
- **Vite Build** — Rask utvikling og optimaliserte bygg

## 🛠️ Teknologi

- **Vanilla JavaScript** — Ren JS, ingen rammeverk
- **HTML5** — Semantisk markup
- **CSS3** — Typografi-drevet styling
- **Vite** — Byggeverktøy og dev-server

## 🚀 Kom i Gang

```bash
npm install
npm run dev
```

Åpne `http://localhost:5173`
