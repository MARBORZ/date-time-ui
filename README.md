# Date & Time UI

Shows date and time as a code object. Vanilla JavaScript experiment with typography.

## 🎨 About

Displays current date/time in a code-like format. Just playing with JavaScript Date and monospace fonts. No frameworks.

## ✨ Features

- **Real-time Clock** — Live date and time updates
- **Code-Style Display** — Monospace, object-like presentation
- **Vanilla JavaScript** — No frameworks or libraries
- **Minimal Design** — Typography-focused UI
- **Vite Build** — Fast development and optimized builds

## 🛠️ Tech Stack

- **Vanilla JavaScript** — Pure JS, no frameworks
- **HTML5** — Semantic markup
- **CSS3** — Typography-driven styling
- **Vite** — Build tool and dev server

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:5173`

## 🎓 Learning Goals

- JavaScript Date API handling
- Clean state → render flow patterns
- Typography-driven UI design
- Vanilla JavaScript without frameworks

## 📜 Scripts

- `npm run dev` — Start dev server
- `npm run build` — Build for production
- `npm run preview` — Preview production build
